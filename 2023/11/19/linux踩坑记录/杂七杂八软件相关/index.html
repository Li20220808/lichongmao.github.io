杂七杂八软件相关

2023-11-18

杂七杂八软件相关

1.libreoffice设置中文：

终端输入 yay -q libreoffice

搜索相关包，找到中文语言包进行安装

2.firefox无条件开启网页 Toggle reader view （阅读模式）

安装扩展插件：Activate Reader View

3.LaTeX相关

使用kile前端结合 arch包 texlive-latex

4.clash for windows 

1.安装好后使用 sudo chmod +x /$(填clash安装路径)/cfw 赋予执行权限

如：sudo chmod +x /opt/clash-for-windows-bin/cfw

2.点 Service Mode 安装模块，否则无法开启 TUN Mode（全局代理）

3.小地球亮起表示安装成功，则可以启动 TUN Mode

5.arduino在linux上无法上传，显示：Permission denied

https://arduino.stackexchange.com/questions/54098/arduino-ide-permision-denied-for-upload-ubuntu#54101
运行命令：sudo chmod a+rw /dev/ttyUSB0

其中 ttyUSB0 是开发板连接的usb端口

6.Octave编写代码时尽量不要用中文

如遇中文乱码，选择GBK格式。右键文本文件“在文本编辑器中打开”，即可换格式

7.安装 win11 跳过登录微软帐号

首次安装在欢迎界面按 Shift + F10 调出控制台，输入 oobe\BypassNRO.cmd 重启后即可

另 virtualbox 设置 win11 显示：不要开3d加速；显卡控制器选 VBoxSVGA 另外记得安装 addition 增强iso

（在学校 正版软件管理平台 下载的软件需要同时下载激活工具按步骤[登录学生帐号等]进行激活）  

8.linux将一个文件夹下的所有文件都设置为普通用户可访问

sudo chmod -R 755 /path/to/your/folder

9.virtualbox正常使用要根据内核版本安装这个包（？）

linux61-virtualbox-host-modules

10.安装fcitx

（1）yay -S fcitx fcitx-qt5 fcitx-qt6

（2）编辑/etc/environment 添加以下字句：

GTK_IM_MODULE=fcitx
QT_IM_MODULE=fcitx
XMODIFIERS=@im=fcitx

重启后输入法生效

11.安装 endeavourOS online

连接国内网络，点击前两个更新镜像（mirror）的选项，endeavourOS mirror那个不用点。mirror更新后即可开始 online install

12.archcraft设置默认启动的桌面环境

etc/sddm.conf 

[Autologin] change the line “Session=openbox” to “Session=bspwm” then save the file and reboot.

12.pulsemeeter

linux下创建虚拟麦克风的软件，似乎只能在刚开机时启动，且关闭一次后无法二次启动。bspwm桌面上会显示软件GUI，但xfce桌面不会。

13.bspmusic & rofi_music

删了/Music/下所有原来有的音乐，没想到不小心把archcraft状态栏中的音乐播放器搞坏了，索性不用archcraft(bspwm?)自带的垃圾播放器了。然后下载了KDE的juk音乐播放器搭配软件本身可设置的全局键盘快捷键，并修改了bspwm本身(/home/admin/.config/bspwm/sxhkdrc)的全局快捷键，使音乐播放体验达到了预期。

14. exodiaOS更新相关

问题：yay安装archlinux-keyring时提示无效或已损坏的软件包

网上找到的都是更新或安装其他软件出现的这个错误，更新密钥后就好了。但我这个是更新密钥出现的，可能这个埃及人搞的发行版有点问题。最后手动安装密钥解决问题：

sudo pacman -U archlinux-keyring--any.pkg.tar.zst

（archlinux-keyring--any.pkg.tar.zst 这个文件到官网手动下载）

15.设置betterlockscreen管理的锁屏背景

betterlockscreen -u 'wallpaper_path'
